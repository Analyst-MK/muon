Muon file manager

A powerful and lightweight open source PHP/JavaScript-based file manager for your web space.

View on GitHub

Manage your website from the browser

You don’t need to install anything to manage your web space!

Muon works as a web-app and its simple, fast loading and responsive interface is optimized for modern browsers for desktops, tablets and smartphones.

Everything you need

What else should a file manager do?

  • Browse folders, display sizes, permission and tree structure.
  • Move, copy, delete, rename your files and create new ones.
  • Zip/unzip archives — this requires PHP-Zip extension on your server.
  • Upload multiple files, according to server’s limitations.
  • Built-in text editor; yeah, Muon can edit text files too! This tool is one of the most helpful to manage your site’s pages (HTML, PHP), scripts (JS) and stylesheets (CSS).

All in one file

Less than 100kB, plain PHP, HTML and JavaScript!

Muon’s code is compressed in a single <100kB PHP page (muon.php) and it’s ready to work out-of-the-box. It only requires a PHP (version 5 or newer) interpreter on the server, that is included by default in the majority of web hosting plans. All settings, including login credentials, are stored at the beginning of the file, and of course hidden to visitors, thus no database is required.

Actually, Muon would be too large for a Commodore 64's memory, but if you're using a newer machine you should definitely have no problems ;)

Get Muon now!

It's not just free, it's open source!

You can get muon.php (as well as the source code) from GitHub, read the installation notes and upload it to your web space. Alternatively, you can copy this code to a PHP file on your server, or directly download it via wget:

wget "https://raw.githubusercontent.com/infoein/muon/main/muon.php"
